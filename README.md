# Elias' dotfiles
This repo contains the dotfiles i use on my personal machines.
I've scoured bits and pieces from [Matt Gaunt](https://github.com/gauntface/dotfiles/)
and [Mathias Bynens](https://github.com/mathiasbynens/dotfiles/). 

## Prerequisites
* ZSH
* NVM
* FFMPEG
* Gifsicle
* Materialshell
* Sublime Text 3
